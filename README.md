# gb-pricing

Сайт для сотрудников с розничными ценами на каталог RM Group.

## Возможности

- Авторизация администраторов и продавцов.
- Приём каталога с ценами по API RM Group (`pricelist.get`).
- Настраиваемая наценка: глобальная, по категории, по товару.
- Приоритет наценок: товар → категория → глобальная.
- Округление итоговой цены (по умолчанию до сотен рублей вверх).
- Автоматическое обновление каталога по cron.
- Автодеплой на хостинг при push в main через `public/deploy.php`.
- Страница продавца с поиском и фильтрами.

## Стек

- PHP 8.2+ (без фреймворков)
- PostgreSQL 16
- чистый HTML/CSS

## Установка

1. Создать облачную БД PostgreSQL 16 в панели хостинга (или локально).
2. Заполнить `src/config.php` реквизитами БД.
3. Выполнить `sql/schema.sql` в базе.
4. Запустить `public/install.php` и создать первого администратора.
5. В панели хостинга настроить cron на выполнение `php /path/to/project/cron/sync.php` каждые 5–15 минут.
6. Зайти в `/admin/`, указать URL и токен API RM Group, задать наценку по умолчанию.

## Роли

- `admin` — настройки API, управление наценками и учётными записями продавцов.
- `seller` — просмотр каталога с розничными ценами.
