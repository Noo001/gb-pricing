# AGENTS.md — GB Pricing

## Проект

Сайт для сотрудников с розничными ценами на каталог RM Group.

- Репозиторий: https://github.com/Noo001/gb-pricing
- Папка: `C:\repos\gb-pricing`
- Стек: PHP 8.3 (без фреймворков), PostgreSQL, чистый HTML/CSS
- Хостинг планируется: Бегет, поддомен `pricing.gadget-bar.ru`

## Что уже сделано

- Авторизация: роли `admin` и `seller`.
- Админка: настройки API, управление пользователями, наценки, лог синхронизаций.
- Приём каталога по API RM Group (`pricelist.get`) через `cron/sync.php`.
- Приоритет наценок: товар → категория → глобальная.
- Округление цены вверх до заданного шага (по умолчанию 100 ₽).
- Страница продавца с поиском и фильтрами.
- Локальное тестовое окружение: `docker-compose.test.yml` + мок API.

## Что нужно доделать

1. Разобраться с хостингом:
   - где сейчас DNS `gadget-bar.ru` (Бегет или другой регистратор/сервер);
   - настроить поддомен `pricing.gadget-bar.ru` на Бегете;
   - привязать document root к `public/`;
   - включить PHP 8.3 и SSL.
2. Создать облачную PostgreSQL на Бегете, выполнить `sql/schema.sql`.
3. Прописать реквизиты БД в `src/config.php` (или через env в панели Бегета).
4. Создать первого админа через `public/install.php`, затем удалить файл.
5. Настроить cron на `php /path/to/gb-pricing/cron/sync.php` каждые 5–15 минут.
6. Уточнить у заказчика формат авторизации в API RM Group (URL, заголовок токена).
7. После получения реального токена проверить синхронизацию и цены.

## Структура

```
public/          # web root
src/             # PHP-логика, конфиг, авторизация
cron/            # скрипт синхронизации
sql/             # схема БД
docker/test/     # тестовое окружение
docker-compose.test.yml
```

## Запуск локального теста

```bash
cd /mnt/c/repos/gb-pricing
docker compose -f docker-compose.test.yml up -d
```

Сайт доступен на `http://localhost:8081` внутри WSL.

## Важно

- `src/config.php` в `.gitignore`. На сервере правится вручную.
- GitHub OAuth token (`gho_...`) был случайно показан в предыдущей сессии — рекомендуется отозвать его в GitHub.
